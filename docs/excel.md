1. **VLOOKUP or XLOOKUP to merge data**:
2. VLOOKUP: `=VLOOKUP(A2, 'Sheet2'!A:B, 2, FALSE)`
3. XLOOKUP: `=XLOOKUP(A2, 'Sheet2'!A:A, 'Sheet2'!B:B)`
4. **Difference between absolute and relative references**:
5. **Relative**: `A1`, adjusts when copied.
6. **Absolute**: `$A$1`, does not adjust.  
   *Use absolute when referencing fixed data.*
